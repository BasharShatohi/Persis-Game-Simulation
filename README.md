**A Java-based implementation of the Persian board game "Persis" (برسيس) featuring intelligent move generation, board state management, and AI decision-making using search algorithms.**


## ✨ Features

- Full simulation of the traditional **Persis** board game
- Object-Oriented Design with classes for `Board`, `Piece`, `Player`, and `GameController`
- Intelligent move validation and rule enforcement
- Dice rolling simulation with probability handling
- Turn-based gameplay with human vs AI or AI vs AI modes
- Board visualization and state tracking
- Search algorithms for optimal move selection

## 🛠 Technologies & Concepts Used

- **Language**: Java
- **Paradigm**: Object-Oriented Programming + Game AI
- **Key Topics**:
  - Search Algorithms (Minimax, Alpha-Beta Pruning)
  - Game Theory
  - State Space Representation
  - Heuristic Evaluation Functions

## 🎯 Core Classes

- `Board`: Manages grid, pieces, and game state
- `Piece`: Represents game pieces with symbols and owners
- `Player`: Human or AI player with strategy
- `GameController`: Main logic, move handling, and dice mechanics
